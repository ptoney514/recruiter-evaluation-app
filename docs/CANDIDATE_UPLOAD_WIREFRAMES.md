## Error Handling

### Upload Errors

**File too large:**
- Show error badge on file item
- Don't upload that file
- Continue with valid files

**Parsing failed:**
- Resume uploads but text extraction fails
- Save with resume_text = null
- Show warning: "Could not parse resume - [Edit Manually]"
- Can still evaluate with manual entry

**Storage error:**
- Retry upload (3 attempts)
- If fails: Show error, don't save candidate
- User can retry or skip that file

---

### Evaluation Errors

**AI API failure:**
- Show error on candidate card
- Status remains "pending"
- Can retry evaluation
- Don't charge for failed attempts

**Performance Profile missing:**
- Warning in confirmation modal
- "Evaluation quality will be lower without complete profile"
- User can proceed or complete profile first
